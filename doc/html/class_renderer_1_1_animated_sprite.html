Основной конструктор анимированного спрайта

Данный конструктор создаёт анимированный спрайт на основе шейдерной программы и субтекстуры из текстурного атласа.

Аргументы
[in]textureТекстура (как правило текстурный атлас)
[in]initialSubTextureНазвание субтекстуры из текстурного атласа
[in]shaderProgramШейдерная программа
[in]positionПозиция спрайта
[in]sizeРазмер спрайта
[in]rotationAngleУгол поворота спрайта
